# Communi.tube (communitube)

Communities for Youtube

---
# Instruções

## Instale as dependências
```bash
npm install
```

### Executar em modo de Desenvolvimento (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Lint nos arquivos
```bash
npm run lint
```

### Build para produção
```bash
quasar build
```
